# Sauce Demo Manual Testing Project

## Project Overview
This project contains manual testing work done on the Sauce Demo web application. It includes test cases and bug reports created during functional testing.

## Application URL
https://www.saucedemo.com/

## Scope of Testing
Functional testing was performed on the main features of the application such as login, products, cart, checkout, and order flow.

- Navigation menu
